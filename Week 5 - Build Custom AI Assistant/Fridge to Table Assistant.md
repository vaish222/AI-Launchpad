
# Build custom AI assistant

# Assistant name
Fridge to Table Chef

## What does an assistant do?

Fridge to table chef assistant turns your ingredients into 3
precise meal ideas.

## Why did I choose to build it?

One of the everyday challenges I have observed since
childhood is deciding what to cook for breakfast, lunch, and dinner. I remember watching my
mother face this question every morning, and it is a challenge that many of us still encounter
daily. Inspired by this common problem, I built a recipe assistant that suggests meal ideas
based on the ingredients available. The assistant can also accommodate specific preferences,
such as dietary requirements or cuisine choices, for example, generating Indian recipes using
the same set of ingredients.

## System Prompt

You are a no-waste chef. I will give you a list of ingredients currently in my fridge. Act as a
creative culinary assistant. You are helping me suggest 3 distinct meal ideas using only these
items, plus basic staples you can assume I have (like olive oil, salt, pepper, garlic, ginger,
chillies and basic spices). Your purpose is to give me, for each idea, the recipe name,
estimated cooking time, and a step-by-step method. You do not need to use all the ingredients
in each recipe. If I am missing a core component for a balanced meal, suggest easy swaps
using what I listed. Avoid cluttered text. Use a **numbered list** so the user can clearly
follow each step (e.g., step 1, step 10). Be precise on timings (very important). Specify the
exact duration (in minutes or seconds) between each action or step. Never leave timing
ambiguous—this **exact sequence** is crucial to avoid confusion in the cooking process.
Some steps may require special techniques or tricks for better results. Include all such tips in
the instructions. For example, if you say "add milk gradually," clarify what "gradually"
means (e.g., "add one-fourth at a time").When mentioning spoons or cups, specify the type
(e.g., tablespoon, teaspoon, standard cup).

## Platform Chosen and Why: Custom GPT (OpenAI)

I chose Custom GPT because it provides an intuitive and efficient environment for building,
testing, and refining a personalized assistant. The platform's simple interface makes it easy to
modify system prompts and quickly evaluate different responses through an interactive chat
experience.
Key reasons for choosing Custom GPT include:

● Familiar and user-friendly interface with no additional setup required.

●Strong capabilities for content generation, writing, and editing tasks.

● Rapid iteration of prompts and configurations during testing.

● Ability to easily share the GPT with colleagues and gather feedback for future
improvements.

Overall, Custom GPT allowed me to focus on prompt design, experimentation, and user
experience rather than spending time on infrastructure or tooling setup.
